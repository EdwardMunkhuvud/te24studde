# Ljudfiler till Linn-sidan

Spela in ett meddelande per fil och lägg filerna i den här mappen.

Filnamnen måste matcha meddelandenas id:

- `001.m4a`
- `002.m4a`
- `003.m4a`
- ...
- `050.m4a`

Sidan spelar upp `/linn-ljud/001.m4a` för meddelande 001, `/linn-ljud/002.m4a` för meddelande 002 och så vidare.

Tips: M4A fungerar bra i Safari på iPhone. Behåll bara samma numrering som meddelandena.
