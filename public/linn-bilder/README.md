# Bilder till Linn-sidan

Lägg bilder på dig i den här mappen.

Enklast: döp dem så här:

- `001.jpeg`
- `002.jpeg`
- `003.jpeg`
- ...

Sidan letar automatiskt efter bilder från `001` till `030` och testar formatet:

- `.jpeg`

Du behöver alltså inte ändra HTML-filen om du lägger in bilder som `001.jpeg`, `002.jpeg`, `003.jpeg` och så vidare.

Som standard letar sidan efter `001.jpg` till `030.jpg`. Vill du ha fler bilder kan du öka siffran `30` i `photoFiles` längst ner i `public/tilllinnochendastlinn.html`.

Tips för iPhone: stående bilder funkar bäst, gärna ungefär 4:5-format. Sidan beskär snyggt så bilden fyller ramen.
